# ✅ My Todo — 類 Microsoft To-Do 的離線待辦清單

一個功能完整、無廣告、無追蹤、零依賴的待辦事項 Web App，靈感來自 Microsoft To-Do。
純前端實作，資料存在瀏覽器 localStorage，可部署到 GitHub Pages。

## ✨ 功能一覽

### 智慧清單
- **☀️ 我的一天** — 每日規劃，建議系統自動推薦重要 / 有到期日的任務
- **⭐ 重要** — 所有標記星號的任務
- **📅 已規劃** — 所有設有到期日的任務
- **📋 所有任務** — 全部未完成任務
- **✅ 已完成** — 歸檔已完成的任務

### 自訂清單
- 無限建立自訂清單
- 支援重新命名 / 刪除
- 每個清單自動計數

### 任務功能
- 📝 建立任務 & 子任務（步驟）
- ⭐ 標記為重要
- 📅 到期日（快捷選擇：今天 / 明天 / 下週 / 自訂）
- 🔔 提醒通知（瀏覽器 Notification API）
- 🔁 重複任務（每天 / 工作日 / 每週 / 每月 / 每年）
- 🏷️ 分類標籤（6 種顏色）
- 📎 附加檔案
- 📝 備註
- ✅ 完成 / 取消完成

### 操作
- 🔍 全局搜尋（標題 / 備註 / 步驟）
- ⇅ 排序（名稱 / 到期日 / 建立日 / 重要性）
- 💡 「我的一天」建議面板
- 🌓 淺色 / 深色主題切換
- 📤📥 匯出 / 匯入 JSON 備份
- ⌨️ 快捷鍵：`Ctrl+N` 新增、`Ctrl+/` 搜尋、`Esc` 關閉

## 🚀 使用方式

### GitHub Pages 部署
1. Fork 這個 repo
2. Settings → Pages → Source: `main` branch, `/root`
3. 訪問 `https://你的帳號.github.io/my-todo/`

### 本地使用
直接用瀏覽器開啟 `index.html`
